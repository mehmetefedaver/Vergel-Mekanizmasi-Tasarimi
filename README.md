# Vergel Mekanizması Tasarımı ve Animasyonu ⚙️

Bu depo, dairesel hareketi hızlı dönüşlü doğrusal harekete çeviren bir **Vergel Mekanizması (Quick Return Mechanism)** tasarımını içermektedir. Proje, mekanik tasarım prensiplerini ve sistemin hareket dinamiklerini görselleştirmek amacıyla hazırlanmıştır.

## 🛠️ Teknik Detaylar
- **Tasarım:** SolidWorks üzerinden 3D olarak modellenmiştir.
- **Mekanizma Tipi:** Whitworth / Slotted Link (Hızlı Dönüş Mekanizması).
- **Format:** Evrensel kullanım için **.STEP** formatında paylaşılmıştır.

## 📁 Proje Dosyaları
Proje dosyalarına ulaşmak için yukarıdaki dosya listesinden STEP klasörüne göz atabilirsiniz. Bu dosya; AutoCAD, Fusion 360, CATIA ve diğer tüm CAD yazılımları ile uyumludur.
